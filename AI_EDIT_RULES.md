# AI Edit Rules
Encoding: All files MUST be saved in UTF-8.

String Handling: For non-ASCII characters (Traditional Chinese), use raw strings or ensure the script explicitly sets encoding='utf-8' when performing I/O.

Change Scope: Do not rewrite the entire file unless requested. Use "Search and Replace" style blocks.

Comments: Keep existing comments. If you must add Chinese comments, ensure they are UTF-8 compliant.
